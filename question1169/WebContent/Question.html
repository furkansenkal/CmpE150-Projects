Question 1:

Find Pairs (100 points)

Please read important information first.

Description

You are expected to implement the function find_pairs(lst,x) which takes a list lst and an integer x.
This function is supposed to find all pairs in the list whose sum is equal to x and print them as tuples. A couple of rules:

  • A number can not be a pair with itself (itself meaning the number at the same index, check examples 3 and 4). Pairs should consist of different numbers (different meaning at different indexes).
  • You can print either (a,b) or (b,a) for one pair (check example 1).
  • You should print all the pairs even if they are the same (same meaning same value, not same index). (check example 2).
You can see that input taking part is handled for you. There are two inputs, the first one will always be an integer which is the desired sum. The second one will always be a list of integers which you will find the pairs on.

Check the examples for further clarification. Keep in mind that we will be grading your code not just based on these examples, but other cases as well, so try to write code which can handle all possible cases.

Hint: If you start from the beginning of the list, your outputs should match ours (please do). If not, as long as you get all the pairs, you will get full points (we will handle it in objection).

Warning: You are not allowed to use any imports and any topics that haven't been covered this semester.
