Quiz 2: Remove The Element (100 points)

Please read important information first.

Description

Write a Python program that removes all occurences of an element from a list, modify other elements and then prints this list. For the input, first read an integer N where N>=0, then read N more integers and store them in a list. Finally read one more integer X. You should remove all occurences of X from the list and multiply the remaining integers in the list with X. Then print the final list.
We have already gave you the print statement at the end of the code (print(final_list)). So you shouldn't write any print statement. You should make necessary adjustments so that the final_list variable has the correct list to print. If N=0 or all the elements are removed, then you should print an empty list such as [].

Assumption: You can assume that all the inputs will be proper.
Warning: Do not print any additional/unnecessary things as output. Your outputs should exactly match with the test case outputs.

Examples:

9
7
7
10

7
5

Explanation for the second input-output:
Number of elements in the list is N = 7, then our initial list is: initial-list = [5,20,15,20,25,50,20] , finally X = 20.
Delete X from the list ([5,15,25,50]) and multiply remaining elements with X: [100, 300, 500, 1000].
