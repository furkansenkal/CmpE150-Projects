Question 3: Tiles (100 points)

Please read important information first.

Description

You are recommended to solve this question last.

Given a board configuration in a string and maximum number of actions that you can take, find the tile actions that bring the given board configuration to the target board, where target_board = '123\n456\n789'. You will implement find_actions(given_board, max_actions) function. Here are the guidelines:

  • given_board is a string that contains 3x3 board with unique digit-tiles and a new-line at the end of each row such as given_board = '416\n759\n283'.
  • max_actions correspond to the maximum length of the solution (sequence of actions).
  • solution will be list of action strings.
  • action string should have the following form: 'move <tile> down' or 'move <tile> right'.
  • note that when a tile is moved down, the tile below will be automatically moved up.
  • similarly when a tile is moved right, the tile on the right will be automatically moved left.

Submit your code with clear and understandable comments even though your code does not work - it will be possible to get partial grade from this question if your instructor finds out you were in the right direction.
