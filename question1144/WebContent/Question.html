Quiz 9: Recursive Summation of Nested Lists (100 points)

Please read important information first.

Description

In this quiz you are expected to calculate the summation of all integers in a list recursively. You will write the sum_list(lst) function which takes one parameter lst as a list and returns the summation of all integers in lst as an integer.

The input list can be a nested list (the list can have multiple lists as elements). The lists can be empty. The lists only include integers and strings. You should only take the summation of integers and skip any string.

We have already provided you with the input part. eval(input()) converts the input string to list. Please do not write the input part again.

Your code will be tested with 10 test inputs of 10 points each.

  • 1 input will be a simple list (not nested) of integers.
  • 2 inputs will be a simple list (not nested) of integers and strings.
  • 5 inputs will be nested list of integers.
  • 2 inputs will be nested list of integers and strings.

Warning 1: You are not allowed to use any kind of loops. Non-recursive implementations will be detected by manual inspection and get no points. For the purposes of this question, solving it iteratively and then calling another function recursively does not count as a recursive implementation.

Warning 2: You are not allowed to use global variables, imports and topics that haven't been covered this semester.

Examples:

INPUT OUTPUT
[2,3,2,1,4] 12
[1,2,[3,4,5],0,[6,[8,7],9,10]] 55
[[1],'a',[[3,'4',5]],[6,[7,8],9],3,'ab'] 42
[['ab']] 0
