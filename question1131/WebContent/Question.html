Quiz 7: Words and Numbers in a File (100 points)

Please read important information first.

Description

In this quiz you will read a word (string) from the user. You will search this word in 'dataset.txt' file which is provided inside src folder which is the same folder as your Main.py file. In 'dataset.txt' file there will be several lines and in each line there will be numbers and words separated by spaces. All numbers will be positive integers and all words will only consist of lowercase letters. We want you to find lines that contain the input word and print the summation of the numbers found in those lines. If the input word is not in 'dataset.txt', then print 0.

A word can be found on more than one line. A word can be found more than one time on the same line.

We have already provided you with the input part. Please do not write the input part again.

Examples:

INPUT OUTPUT
comprehensive 8180
holiday 3283
paradox 11406
hello 0

Explanation for the input 'comprehensive': The word 'comprehensive' occurs on 4 different lines in the dataset.txt file, line 11-24-31-32. Summation of the numbers on those lines is 8180.

Explanation for the input 'holiday': The word 'holiday' occurs on 1 line in the dataset.txt file, line 21. Summation of the numbers on line 21 is 3283.
