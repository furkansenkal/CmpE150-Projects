Quiz 4: Repeating Digits (100 points)

Please read important information first.

Description

Write a program that takes an integer number from the user and prints a dictionary.

If the input number has more than 5 digits (if the input number is greater than 99999), then your final dictionary should have the digits of the input number as keys and the counts of those digits as values. You should print the digits and their counts in ascending order of digits.

If the input number has less than 6 digits (if the input number is less than 100000), then your final dictionary should have the most frequent digit as the key and its count as the value. So the final dictionary will have only one key-value pair. For this situation, you can assume that there will always be a unique most frequent digit.

We have already provided you with the input and printing part. In the printing part, we iterate through 0 to 9 and print the digit and its count to the console if the digit is in the dictionary. This printing part ensures that the output is printed in the ascending order of digits. So, you don't need to keep digits in ascending order in your dictionary. Note that in order for the printing part to print the results, your dictionary's keys must be integers.

Examples:

INPUT OUTPUT
135395 1 -> 1
3 -> 2
5 -> 2
9 -> 1
0556245070754 0 -> 2
2 -> 1
4 -> 2
5 -> 4
6 -> 1
7 -> 2
12883 8 -> 2
